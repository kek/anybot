# Anybot

## Future commands

### `!save <name> <program>`
Save a program and run it. The code will be run again at restart.

### `!delete <name>`
Removes a program so that it won't be run at next restart.

### `!list`
List saved programs.

### `!show <name>`
Shows the code for a saved program.

### `!help`
Display help for these commands.
